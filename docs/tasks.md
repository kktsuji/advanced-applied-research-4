#### 2.2 代表的な空間フィルタとその周波数特性

##### ボックスフィルタ（平均化フィルタ）

- [x] フィルタカーネルの定義（均一な重み）
- [x] 数式表現
  - 1次元：$h[n] = \frac{1}{N} (n=0,1,...,N-1)$
  - 2次元：$h[m,n] = \frac{1}{M \times N}$
- [x] 周波数応答特性
  - sinc関数の形状
  - サイドローブの存在
  - カットオフ周波数の急峻性

##### ガウシアンフィルタ

- [x] フィルタカーネルの定義（ガウス分布）
- [x] 数式表現
  - 1次元：$h[n] = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{n^2}{2\sigma^2}}$
  - 2次元：$h[m,n] = \frac{1}{2\pi\sigma^2} e^{-\frac{m^2+n^2}{2\sigma^2}}$
- [x] 周波数応答特性
  - ガウス関数（周波数領域でもガウス分布）
  - 滑らかなロールオフ特性
  - σとカットオフ周波数の関係

##### イプシロンフィルタ（エッジ保存フィルタ）

- [x] フィルタの原理（画素値の差に基づく適応的重み）
- [x] 数式表現
  - 重み関数：$w(x) = \frac{1}{1 + (\frac{|x|}{\epsilon})^2}$
  - 出力：$y[m,n] = \frac{\sum_{i,j} w(I[i,j] - I[m,n]) \cdot I[i,j]}{\sum_{i,j} w(I[i,j] - I[m,n])}$
- [x] 周波数応答特性
  - 信号依存性（非線形フィルタ）
  - エッジ部分での高周波保存特性
  - 平坦部でのローパス特性
  - εパラメータと周波数特性の関係

##### フィルタ比較

- [x] 各フィルタの周波数応答の比較
- [x] エッジ保存性能の違い
- [x] ノイズ除去性能とエッジ保存のトレードオフ

#### 2.3 ノイズリダクション手法の分類と概要

- [x] 空間フィルタリング手法の概要
  - 線形フィルタ（ボックス、ガウシアン）

- [x] 周波数領域フィルタリング手法の概要

  - ウェーブレット変換ベース

- [x] 適応フィルタリング手法の概要
  - ウィナーフィルタ、非局所平均、BM3D等
