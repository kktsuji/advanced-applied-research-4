# 要件定義

## レポートの目的

- 九州工業大学 博士課程の社会人対象科目「特別応用研究Ⅳ（Special Research for Applications IV）」向けのレポート。
- 授業の概要
  - 社会人学生が職場でこれまでに経験してきた実務的・研究内容に関して教員とディスカッションを行う
  - 学問的理解を深め、職場での問題を提起し、問題解決を図る
  - この体験を通じ、工学上の諸問題に対する解決能力と実践能力を高めることを目的とする

## レポートの構成

- タイトル：ハードウェア実装制約を考慮したデジタル画像に対するノイズリダクション画像処理技術
- 議論する課題
  - デジタル画像処理におけるノイズリダクション技術ではフィルタリング手法が一般的
  - ハードウェア実装制約を考慮した場合、計算リソースや消費電力の制限が存在
  - これらの制約下で効果的なノイズリダクション手法の開発が求められる
- 主要なセクション
  - デジタル画像処理の基礎
    - 画素値、カラー空間、画像フォーマットの説明
  - ノイズリダクション技術の概要
    - 空間フィルタリング、周波数領域フィルタリング、適応フィルタリングの説明
  - ハードウェア実装制約とその影響
    - 計算リソースの制限
      - メモリ制約（ラインバッファにより1行ずつ読み込まれる）
      - 処理速度の制約（リアルタイム処理の必要性）
  - 手法
    - 画像縮小＋小サイズフィルタによる多重解像度アプローチ
      - 画像を縮小し、小さなフィルタを適用してノイズを低減
      - 各解像度で高周波成分を抽出し、最終的に統合
      - これにより、大きなフィルタを使用せず、全周波数帯域で効果的なノイズリダクションを実現
    - ハードウェア実装の工夫
      - 横方向：ラインバッファ上で1次元のフィルタを使用
      - 縦方向：IIRフィルタを使用することで必要なラインバッファを削減
